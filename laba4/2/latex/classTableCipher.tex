\doxysubsection{Подробное описание}
Шифрование методом маршрутной перестановки 

Класс \mbox{\hyperlink{classTableCipher}{Table\+Cipher}} предоставляет методы для зашифрования и расшифрования текста с использованием табличного шифра. Ключ для зашифрования и расшифрования должен быть положительным целым числом.
